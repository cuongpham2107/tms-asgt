# Plan: Auto-checkpoint cho xe thuê ngoài + filter loại xe trên Trip list

## Phần A: Tự động tạo checkpoint khi gán trip cho xe thuê ngoài

### Logic

Khi tạo trip (tại 4 điểm: AssignTransport, BulkAssign, OrdersTable inline edit, CreatesOrderTransportCards), nếu `vehicle->type === VehicleOwnerType::Rent` thì tự động tạo **toàn bộ** checkpoint (Started, ArrivedPickup, LeftPickup, ArrivedDelivery, Completed) với `km_reading = null` để điều hành vào điền sau.

### Checkpoint pattern (theo đúng cấu trúc CheckpointFactory hiện tại)

- **Started, ArrivedPickup, LeftPickup**: mỗi order trong trip → 1 checkpoint
- **ArrivedDelivery, Completed**: mỗi delivery point của từng order → 1 checkpoint

Tất cả checkpoint được tạo với:
- `km_reading = null` (điều hành sẽ nhập)
- `occurred_at = now()` (giá trị tạm, điều hành sửa lại)
- `driver_id` & `shift_id` từ trip
- `order_id` & `delivery_point_id` theo đúng order/delivery point

### Files sửa (4 files, thêm logic sau `Trip::create`)

| File | Vị trí thêm |
|------|------------|
| `AssignTransportAction.php` | Sau `Trip::create()`, trước `$record->update()` |
| `BulkAssignTransportAction.php` | Sau vòng lặp `foreach` gán orders |
| `OrdersTable.php` | Sau `Trip::create()` trong inline edit |
| `CreatesOrderTransportCards.php` | Sau `Trip::create()` |

Logic chung (extract ra 1 helper static method để tránh lặp code):

```php
// Trong CreatesOrderTransportCards (trait đã được cả 4 file dùng) thêm method:
public static function createCheckpointsForExternalVehicle(Trip $trip, Collection $orders): void
{
    $vehicle = $trip->vehicle;
    if ($vehicle?->type !== VehicleOwnerType::Rent) {
        return;
    }

    $driverId = $trip->driver_id;
    $shiftId = $trip->shift_id;
    $now = now();

    foreach ($orders as $order) {
        // Trip-wide checkpoints (per order)
        foreach ([CheckpointType::Started, CheckpointType::ArrivedPickup, CheckpointType::LeftPickup] as $type) {
            TripCheckpoint::create([
                'trip_id' => $trip->id,
                'order_id' => $order->id,
                'checkpoint_type' => $type,
                'occurred_at' => $now,
                'km_reading' => null,
                'driver_id' => $driverId,
                'shift_id' => $shiftId,
            ]);
        }

        // Delivery checkpoints (per delivery point)
        foreach ($order->deliveryPoints as $dp) {
            foreach ([CheckpointType::ArrivedDelivery, CheckpointType::Completed] as $type) {
                TripCheckpoint::create([
                    'trip_id' => $trip->id,
                    'order_id' => $order->id,
                    'delivery_point_id' => $dp->id,
                    'checkpoint_type' => $type,
                    'occurred_at' => $now,
                    'km_reading' => null,
                    'driver_id' => $driverId,
                    'shift_id' => $shiftId,
                ]);
            }
        }
    }
}
```

**Lưu ý**: `CreatesOrderTransportCards` là trait được cả 4 file `use`. Thêm static method vào trait này giúp tránh lặp code.

Cần thêm `use` statements: `use App\Enums\CheckpointType;`, `use App\Enums\VehicleOwnerType;`, `use App\Models\TripCheckpoint;`
