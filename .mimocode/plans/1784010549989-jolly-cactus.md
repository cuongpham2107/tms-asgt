# Plan: Trang tổng hợp ca trực lái xe hàng ngày

## Yêu cầu
Một Filament page mới với:
1. **Widget tổng hợp** theo điểm trực (BN/TN/NBA): tổng lái, đi làm (TTL/X/2/Y/2), nghỉ
2. **Table chi tiết** từng lái: điểm trực, tên, xe, ca, số chuyến, km CH, km KH, tổng km, giờ bắt đầu/kết thúc ca

Khung giờ: 8h sáng nay → 8h sáng mai.

## Cấu trúc file

### `app/Filament/Pages/DriverDutyReport.php`
Trang chính, có `getHeaderWidgets()` và 1 bảng dữ liệu.

### `app/Filament/Widgets/DriverDutySummaryWidget.php`
Widget bảng tổng hợp theo điểm trực.

## Logic dữ liệu

### Widget tổng hợp
```php
$drivers = User::role('driver')->where('is_active', true)->get();
// Group by station (TN/BN/NBA)
// For each driver, check if they have an active shift (start_time <= now <= end_time IS NULL)
// Count: TTL (full), X/2 (morning_half), Y/2 (night_half), X (other), nghỉ (no shift)
```

### Table chi tiết
```php
// Drivers có shift trong khoảng [8h today, 8h tomorrow]
$shifts = DriverShift::whereBetween('start_time', [$from, $to])
    ->orWhereNull('end_time')
    ->with('driver', 'trips', 'trips.vehicle')
    ->get();
// Hoặc query shifts overlap với time range
// Join với vehicles qua trips
// Tính số chuyến = trips count
// Km từ shift.trip hoặc shift.total_km_*
```

## Data mapping cho table
| Cột | Nguồn |
|-----|-------|
| STT | index |
| Điểm trực | `user.station` |
| Lịch lái xe | `user.name` |
| Xe chạy | `vehicle.plate_number` (+ "đảo lái" nếu trip swap) |
| Ca | `shift.shift_type` label |
| Số chuyến | `count(trips)` |
| Số Km CH | `shift.total_km_loaded` |
| Số Km KH | `shift.total_km_empty` |
| Tổng KM | `shift.total_km` |
| Bắt đầu ca | `shift.start_time` |
| Kết thúc ca | `shift.end_time` |
