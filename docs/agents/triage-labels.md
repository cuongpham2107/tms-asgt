Triage label vocabulary

Canonical roles and default label names used by skills:

- needs-triage — `needs-triage` (maintainer needs to evaluate)
- needs-info — `needs-info` (waiting on reporter for more information)
- ready-for-agent — `ready-for-agent` (fully specified; AFK agent can pick up)
- ready-for-human — `ready-for-human` (needs human implementation)
- wontfix — `wontfix` (will not be actioned)

Notes

- If your repository uses different label names, edit this file to map the canonical roles to your labels.
- The `triage` skill will apply labels exactly as written here.
